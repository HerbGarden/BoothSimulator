1. 실행파일을 실행합니다.

2. 도면의 전체를 몇 X 몇 (m 단위입니다.)인지 입력해주세요. 
   // 입력하지 않으시면 오류가 납니다.

3. 부스의 크기를 정하여 make 버튼을 누른 뒤 도면 화면에서 좌클릭하면 해당부분을 
  가장 왼쪽 위로 가정하여 부스가 설정됩니다.

4. 부스와 부스 정보를 입력하시고, Entranc/Exit을 이용하여 입장하는 부분과
  나가는 부분을 각각 설정하세요. 하나만 설치하셔도 됩니다.

5. Preference에서 불러오기로 부스 이름을 불러온 뒤 
  해당 부스의 우선순위를 입력하여 주세요. 부스가 5개 이상이면 좌 우 화살표를 클릭하여 설정 가능합니다.
  화면을 이동하기 전에는 저장버튼을 꼭 눌러주세요.

6. 부스를 선택하고 우측 상단의 뷰를 이용하여 부스를 삭제하거나 정보를 변경 할 수 있습니다.
   부스를 우클릭해도 정보를 변경합니다.

7. 맵을 저장하거나 불러올 수 있습니다.

8. Start 버튼에서 최대로 투여할 객체의 갯수를 입력하세요.
   //3,4번을 하지 않으면 시뮬레이션이 실행되지 않습니다.

9. 6번의 설정이 끝나면 시뮬레이션을 시작합니다. 마우스 좌클릭으로 객체를 하나씩 
 투여 할 수 있으며, 투여할 객체의 수를 입력받지만 객체는 100명으로 제한하였습니다.

10. 우클릭을 한번 누르시면 시뮬레이션을 종료합니다.

11. 그 뒤에 한번 더 누르시면 각 부스 위에 부스에 진입시도하였으나 대기줄이 꽉 차 입장하지 못한 횟수와 
    가장 이동량이 많았던 부분을 보라색으로 맵에 표시해줍니다.
